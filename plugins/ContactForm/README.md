# Contact Form Plugin

Plugin thêm block **Form Liên Hệ** vào trình page builder Puck.

## Tính năng
- Kéo thả block form liên hệ vào bất kỳ trang nào dùng Page Builder
- Cấu hình fields (cơ bản/đầy đủ), tiêu đề, mô tả, nút gửi
- Lưu submissions vào database
- Xem danh sách submissions trong admin

## Cài đặt
Kích hoạt plugin trong quản lý Plugin của admin panel.
