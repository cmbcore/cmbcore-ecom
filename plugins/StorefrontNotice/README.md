# Storefront Notice Plugin

Plugin mẫu cho CMBCORE dùng để verify đầy đủ các contract sau:

- quét `plugin.json`
- đồng bộ `installed_plugins`
- bật/tắt plugin trong admin
- lưu `settings`
- inject admin menu/page từ manifest
- boot hook `theme.head`, `theme.footer`, `product.created`

Plugin này chỉ nhằm mục đích kiểm thử và làm mẫu phát triển plugin mới.
